# CLI Execution
Below command will run analysis and provide output on terminal

```sh
npm start
```
OR
```sh
npm start https://bolster.ai
```
# Testing
```sh
npm test
```

# Linting

```sh
npm run lint
```
OR
```sh
npm run lint-fix # with lint fix
```
